# Mobile & PWA Issues

## Mobile Browser Problems

### GPS Not Working
**Problem**: Location services not working on mobile
**Solution**: 
1. Enable location permissions for browser
2. Use HTTPS (required for GPS)
3. Try different browser (Chrome, Safari, Firefox)

### App Not Installing
**Problem**: "Add to Home Screen" not appearing
**Solution**: 
1. Use HTTPS connection
2. Visit site multiple times
3. Check browser supports PWA
4. Clear browser cache

### Touch Issues
**Problem**: Buttons not responding to touch
**Solution**: 
1. Disable zoom if enabled
2. Clear browser cache
3. Restart browser
4. Try different browser

## PWA Problems

### Service Worker Not Loading
**Problem**: Offline functionality not working
**Solution**: 
1. Check HTTPS connection
2. Clear service worker cache
3. Unregister and re-register service worker

### Update Not Installing
**Problem**: New version not updating
**Solution**: 
1. Close all app tabs
2. Clear cache
3. Restart browser
4. Revisit site

### Offline Mode Issues
**Problem**: App doesn't work offline
**Solution**: 
1. Ensure you visited online first
2. Check service worker is registered
3. Clear cache and reload

## Theme Issues

### Dark Mode Not Working
**Problem**: Theme doesn't switch properly
**Solution**: 
1. Clear localStorage
2. Reset theme in Settings
3. Restart browser

### UI Elements Broken
**Problem**: Buttons or layout broken on mobile
**Solution**: 
1. Check screen orientation
2. Clear browser cache
3. Try different viewport size

## Performance on Mobile

### Slow Loading
**Problem**: App loads slowly on mobile
**Solution**: 
1. Use WiFi instead of mobile data
2. Close other browser tabs
3. Clear cache
4. Restart browser

### Battery Drain
**Problem**: App uses too much battery
**Solution**: 
1. Reduce refresh frequency
2. Close app when not in use
3. Disable background refresh

## Quick Mobile Fixes

### Reset Mobile App
1. Clear browser data for site
2. Remove from home screen
3. Restart browser
4. Re-add to home screen

### Test Different Browser
- **Android**: Chrome, Firefox, Samsung Internet
- **iOS**: Safari, Chrome, Firefox

### Check Permissions
1. Go to browser settings
2. Find site permissions
3. Enable location, notifications
4. Restart browser
